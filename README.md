Just a simple React dashboard backed up by a fake [JSON API](http://beta.json-generator.com/) just for fun.

# Tech

* React
* ES2015
* Webpack
* PostCSS

### To run the app

```sh
$ npm i && npm start
```

This will run the `webpack-dev-server` and you will be able to view the app on `http://localhost:8080`

### To build the app

```sh
$ npm run build
```
